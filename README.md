 Student Management System

A simple student management system developed with C# Console Application.

 Features

- Add Student
- List Students
- Search Student
- Remove Student
- Prevent duplicate student records
- Exception handling with try-catch

Technologies

- C#
- .NET Console Application
- List<T>
- Switch-Case
- While Loop
- Try-Catch
  About

This project was developed to practice fundamental C# programming concepts such as lists, loops, conditions, methods, and exception handling.
